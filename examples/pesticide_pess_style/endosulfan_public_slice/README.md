# Endosulfan public slice (PESS-style)

PESS head-to-head dietary case pack for **Endosulfan** (CAS 115-29-7), mirroring `glyphosate_public_slice`.

- Toxicological reference values (ADI 0.006 mg/kg bw/day, ARfD 0.015 mg/kg bw/day) are the **real governed EFSA OpenFoodTox** values shipped in `defaults/v1`.
- Residue inputs are **illustrative regulatory-screening values** within the observed EU MRL band (`schema_reviewed_not_source_validated`), not row-level EFSA monitoring data.

Regenerate with `build_pess_case_packs.py`. See `limitations.md` and `source_lock.json`.

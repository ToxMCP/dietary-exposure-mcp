# Dietary MCP v0.1.0

Status: stable release draft

Stable `v0.1.0` is reserved until the public release gates in
[`docs/public_release.md`](../public_release.md) are complete. The current
reviewable package is documented in
[`v0.1.0-rc1.md`](./v0.1.0-rc1.md).

The stable release must record independent scientific signoff, OpenFoodTox 3.0
reconciliation, the final third-party data-rights posture, and a clean rerun of
the complete release verifier.
